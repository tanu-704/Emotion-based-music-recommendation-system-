# 🎵 Emotion Based Music Recommendation System

## 📌 About
This project detects real-time facial emotion using Mediapipe + Deep Learning model and recommends songs based on detected emotion.

## 🚀 Features
- Real-time webcam emotion detection
- Face + hand landmark tracking
- Music recommendation via YouTube
- Streamlit web interface

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
